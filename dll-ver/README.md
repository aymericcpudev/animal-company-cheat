# Animal Company Cheat DLL

This project is a C++ DLL intended for injection into Animal Company (Oculus VR, Windows). It provides a framework for implementing cheats such as ESP, speed hacks, and more. 

## Features
- DLL entry point
- Injection stub
- Cheat feature stubs (to be implemented)

## Usage
1. Build the DLL using Visual Studio or MSVC.
2. Inject the DLL into the Animal Company process using your preferred injector.
3. Extend the code with your desired cheat features.
